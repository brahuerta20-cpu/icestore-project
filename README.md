# Ice Store
Análisis de ventas para tienda de videojuegos.

La tienda online Ice comercializa videojuegos a nivel mundial y dispone de información histórica sobre ventas, reseñas de usuarios y críticos, géneros, clasificaciones ESRB y plataformas de lanzamiento. A finales de 2016, la empresa requiere identificar los factores que influyen en el éxito comercial de un videojuego para optimizar la planificación de campañas publicitarias durante 2017.

Este proyecto consiste en analizar los datos históricos de videojuegos disponibles hasta 2016 para identificar patrones, tendencias y variables clave que permitan determinar qué características están asociadas con mayores niveles de ventas. El objetivo es generar información accionable que facilite la selección de títulos prometedores y la asignación eficiente del presupuesto publicitario.

# Herramientas y tecnologías.

- Excel: Archivo CSV.
- Jupyter Notebook.
- Librerías de Python: Pandas, numpy, matplotlib y scipy.
- Procesamiento y manipulación de datos.
- Visualización de datos.
- Pruebas de hipótesis.

# Metodologías y análisis.

- Realicé la limpieza y preparación de los datos, identificando valores faltantes, registros duplicados y posibles inconsistencias.
- Estandaricé formatos de variables y generé métricas complementarias, como las ventas globales por videojuego.
- Exploré la evolución de las ventas por plataforma y género mediante análisis descriptivos y visualizaciones.
- Identifiqué plataformas con crecimiento, estabilidad o declive para determinar cuáles presentaban mayor potencial comercial.
- Analicé la relación entre las calificaciones de usuarios y críticos con el desempeño en ventas.
- Comparé el comportamiento de diferentes mercados regionales (Norteamérica, Europa y Japón) para detectar preferencias específicas.
- Evalué el impacto de las clasificaciones ESRB sobre las ventas y el comportamiento del consumidor.
- Formulé y validé hipótesis estadísticas para determinar si existían diferencias significativas entre grupos relevantes de videojuegos y plataformas.

# Resultados y conclusiones.

Se identificaron los factores con mayor influencia en el éxito comercial de los videojuegos, incluyendo plataformas con mejor proyección de crecimiento, géneros más rentables y la relevancia de las reseñas de críticos en el desempeño de ventas. Los hallazgos permitieron generar recomendaciones estratégicas para la planificación de campañas publicitarias de 2017, facilitando la selección de títulos con mayor potencial de éxito y mejorando la toma de decisiones basada en datos.
